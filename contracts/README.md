## Installation

### Testrpc

```
npm install -g ethereumjs-testrpc
```

### Truffle

```
npm install -g truffle
```

### NPM packages

```
npm install
```

## Compile

```
truffle compile
```

Contract will be compile in `contracts` folder.

## Test

Launch `testrpc` is a new terminal

```
testrpc
```

Then

```
truffle test
```
